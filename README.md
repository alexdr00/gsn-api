# Start development environment
1. Install dependencies 
    > npm i
2. Create `.env` file next to `.env.example`. Then copy everything inside
`.env.example` and paste it in `.env`, then replace all values.

3. Start dev server
    > npm run dev 

# Tests
- Run unit tests
> npm run ut

- Run unit tests (watch mode)
> npm run ut:watch


# Production
### Compile and run
> npm start

### Just compile
> npm run tsc
